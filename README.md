# Bitcoin Design Interview Voting

Community voting board for the AI-Age Bitcoin.Design research interview framework.

## Live app

**https://interview-voting.vercel.app**

## Run locally

```bash
npm install
npm run seed
npm start
```

Open http://localhost:3847

## Deploy

### Vercel (current)

```bash
npx vercel deploy --prod
```
