# Bitcoin Design Interview Voting

Community voting board for the AI-Age Bitcoin.Design research interview framework.

## Run locally

```bash
npm install
npm run seed
npm start
```

Open http://localhost:3847

## Deploy to Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/Aditya-Sarna/bitcoin-design-interview-voting)

Or connect this repo in the [Render dashboard](https://dashboard.render.com/) — it will use `render.yaml` automatically.
